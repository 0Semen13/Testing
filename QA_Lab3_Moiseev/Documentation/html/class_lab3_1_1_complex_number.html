float Lab3.ComplexNumber.Modulus ( )
inline

Функция высчитывания модуля комплексного числа

Returns
Модуль комплексного числа

Данная функция вычисляет значение модуля комплексного числа по формуле: \(\sqrt{(realPart)^2+(imaginaryPart)^2}\).

Код функции выглядит следубщим образом:

public float Modulus() {
float modul = (float)Math.Sqrt((realPart * realPart) + (imaginaryPart * imaginaryPart));
return modul;
}
